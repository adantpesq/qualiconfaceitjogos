Este repositório disponibiliza a planilha de dados utilizada na pesquisa sobre a relação entre qualidade de conformidade técnica e aceitação do usuário em jogos digitais. Os registros foram coletados manualmente em fontes públicas e organizados para possibilitar rastreabilidade, transparência e reprodutibilidade parcial do procedimento analítico. A base reúne os 100 jogos incluídos na amostra, com as variáveis empregadas na análise estatística e os respectivos indicadores de avaliação crítica e aprovação pública.
